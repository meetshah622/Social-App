**Social App**
--------------
*Practical demo for Figma design to react component design traning*
--------------

> Clone the Project
```git clone https://github.com/meetshah622/Social-App.git```

> Go to the project directory
```cd Social-App```

> Install the dependancy
```npm install```

> Run locally
```npm run dev```
